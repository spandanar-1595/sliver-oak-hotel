# Silver Oak Hotel Website

## Project Overview
Silver Oak Hotel is a responsive static hotel website developed using HTML, CSS, and JavaScript. The website provides information about the hotel, available rooms, gallery, and an online booking form.

## Features
- Responsive design for mobile, tablet, and desktop
- Home, About, Rooms, Gallery, and Booking pages
- Room booking form with JavaScript validation
- Navigation menu on all pages
- SEO-friendly HTML (title, meta description, keywords)
- Image gallery with alt text

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## How to Run
1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser or use the Live Server extension.
